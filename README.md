# AI Skills 仓库

这是一个集合了多种专业领域 AI Skill 的仓库。这些 Skill 旨在赋予 AI 助手（如 Claude/Gemini）在特定场景下的专家级能力，通过结构化的工作流、核心框架和行业知识，提升任务完成的质量与效率。

## 🚀 当前 Skill 列表

### 🏗️ 架构与分析 (Architecture & Analysis)

- **[Axiomatic Design (公理设计)](file:///c:/Users/golde/code/skills/axiomatic-design/SKILL.md)**
  - **核心用途**：基于公理设计理论，将自然语言需求转化为高内聚、低耦合的软件架构。
  - **适用场景**：系统架构设计、模块解耦分析、定义严格的接口契约（DbC）。
  - **特点**：结合函数式编程原则，确保架构的科学性与可维护性。

- **[Literate Programming (文学化编程)](file:///c:/Users/golde/code/skills/literate-programming/SKILL.md)**
  - **核心用途**：运用文学化编程哲学，通过“织入（Weaving）”故事的方式对 codebase 进行深度分析。
  - **适用场景**：复杂项目架构审计、生成文学化架构记录（LAR）、像讲故事一样解释代码逻辑。
  - **特点**：强调“代码即散文”，注重叙事连续性与渐进式说明。

### 💼 商业与创业 (Business & Entrepreneurship)

- **[Business Idea Analyser (商业创意分析师)](file:///c:/Users/golde/code/skills/business-idea-analyser/business-idea-analyser/SKILL.md)**
  - **核心用途**：风险投资级别的创意评估。运用 YC 框架、SUCCESs 模型进行压力测试。
  - **适用场景**：商业计划预演、痛点强度分析、市场进入策略（GTM）制定。
  - **特点**：风格专业且严谨，挑战假设，生成全面的战略分析报告。

- **[Startup Helper (创业助力套件)](file:///c:/Users/golde/code/skills/startup-helper/)**
  - **核心用途**：面向初创团队的综合辅助工具集。
  - **包含内容**：涵盖创业策略导航、集成的商业计划书模版、技术顾问、融资路演辅导等多个维度的专业 Skill。
  - **特点**：强调实战落地，特别是针对医疗/视光等垂直行业的商业闭环设计。
