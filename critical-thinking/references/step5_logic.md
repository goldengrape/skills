# 第5步：逻辑强度评估 (Logical Evaluation)

基于《Critical Thinking》第12版 第2、9、10、11章。

## 区分推理类型

### 演绎论证 (Deductive Arguments)

- **目标**：**证明**结论。前提为真 → 结论**必须**为真
- **关键词**："必然"、"肯定"、"绝对"
- 示例："所有人都有一死，苏格拉底是人，所以苏格拉底会死"

### 归纳论证 (Inductive Arguments)

- **目标**：**支持**结论。前提为真 → 结论**很可能**为真
- **关键词**："可能"、"很可能"、"大概"
- 示例："那里乌云很厚，所以大概要下雨了"

### 其他推理类型

- **最佳解释推理 (IBE)**：某解释为真因为它比替代解释更好地说明已知现象
- **权衡利弊**：通过权衡支持和反对的理由做出决定

---

## 评估演绎论证

### 有效性 (Validity)

前提全部为真时，结论**不可能**为假 → 有效。

- 只关乎结构，不关乎前提是否实际为真
- 结论荒谬的论证也可能结构有效

### 可靠性 (Soundness)

有效 + 前提在现实中确实为真 → 可靠。**批判性思维的终极目标。**

### 常见有效形式

| 名称 | 结构 |
|------|------|
| 肯定前件 (Modus Ponens) | 如果P则Q；P；所以Q |
| 否定后件 (Modus Tollens) | 如果P则Q；非Q；所以非P |
| 连锁论证 (Chain) | 如果P则Q；如果Q则R；所以如果P则R |
| 选言三段论 (Disjunctive) | P或Q；非P；所以Q |

## 评估归纳论证

### 强与弱

- **强论证**：前提确实提高了结论为真的概率
- **弱论证**：前提未能显著提高结论为真的概率

### 具体形式的评估标准

#### 类比论证

- 形式：X和Y有属性a,b,c；X有属性d → Y也有属性d
- 评估：相似点越多越相关 → 越强；差异点越多越相关 → 越弱

#### 样本概括

- 形式：样本中X%具有某特征 → 总体中也有X%
- 评估：
  1. **样本大小**：够大吗？
  2. **代表性**：随机抽取？是否有自我选择偏差？

#### 因果推理

- 形式：A发生后B发生 → A导致B
- 评估：排除巧合？排除共同原因？排除因果倒置？（对照实验 > 轶事证据）

## 分析步骤

1. **定性**：论证试图**证明**结论（演绎）还是**支持**结论（归纳）？
   - 不确定时，根据宽容原则视为归纳处理
2. **演绎评估**（若为演绎）：
   - 结构是否有效？（检查肯定后件、否定前件等结构谬误）
   - 前提是否为真？（有效 + 前提真 → 可靠）
3. **归纳评估**（若为归纳）：
   - 概括：样本大小和代表性
   - 类比：是否恰当？关键差异？
   - 因果：混淆相关性与因果性？
   - 最佳解释：是否有更好的替代解释？
4. **综合评级**：逻辑严密 / 有力支持 / 微弱支持 / 逻辑崩塌
