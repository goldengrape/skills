# 第3步：心理与修辞检测 (Cognitive & Rhetorical Check)

基于《Critical Thinking》第12版 第1章与第5章。

## 第一部分：认知偏差 (Cognitive Biases)

检查作者是否表现出以下偏差，或文本是否试图利用读者的这些偏差。

### 信念形成偏差

- **信念偏差 (Belief Bias)**：根据结论的可信度评估论证逻辑强度——同意结论则误以为逻辑有效
- **确认偏差 (Confirmation Bias)**：重视支持既有观点的证据，忽视/拒斥反面证据

### 概率与评估偏差

- **可得性启发法 (Availability Heuristic)**：根据事件在记忆中被回想的容易程度判断频率/可能性
- **消极偏见 (Negativity Bias)**：对负面信息赋予的权重远高于正面信息
- **损失规避 (Loss Aversion)**：避免损失的动力强于获得收益的动力

### 社会与自我偏差

- **内群体偏见 (In-group Bias)**：对所属群体成员给予更优待评价
- **基本归因错误 (Fundamental Attribution Error)**：他人行为归因于性格，自己行为归因于环境
- **服从权威 (Obedience to Authority)**：盲目听从权威指令
- **过度自信效应 (Overconfidence Effect)**：高估自己判断的正确率
- **优于平均错觉 (Better-than-average Illusion)**：认为自己高于平均水平

## 第二部分：修辞手段 (Rhetorical Devices)

修辞力 ≠ 逻辑力。修辞通过情感色彩产生心理影响，而非提供理由。

### 词语情感色彩操纵

- **委婉语 (Euphemisms)**：用正面/中立词淡化负面概念（"附带损害" ← "平民死亡"）
- **粗直语 (Dysphemisms)**：用负面词强化负面联想（"极端分子" ← "反叛者"）

### 规避与贬低

- **闪烁其词 (Weaselers)**：插入词语削弱断言确定性（"高达"、"有的"、"可能"、"据说"）
- **贬抑 (Downplayers)**：降低某人/事的重要性
  - 手段："仅仅是"、"只不过"、引号（他的"学位"）、"虽然…但是…"

### 暗示与联想

- **刻板印象 (Stereotypes)**：基于群体的未经证实的概括假设
- **暗示 (Innuendo)**：用言外之意传达贬义信息
  - *显著提及 (Significant Mention)*：提及不必要的背景信息暗示关联
- **加载问题 (Loaded Questions)**：问题中埋设未经证实的假设

### 夸张与嘲讽

- **嘲笑/讽刺 (Ridicule/Sarcasm)**：通过嘲笑拒绝立场，非反驳论证
- **夸张 (Hyperbole)**：极度夸大，用于煽动情绪

### 伪装成论证的修辞

| 手段 | 描述 |
|------|------|
| 修辞性定义 | 用情感色彩定义术语以表达态度 |
| 修辞性解释 | 貌似解释原因，实则表达态度（"他输了因为没骨气"） |
| 修辞性类比 | 通过比较传达情感评价而非逻辑相似性 |
| 替代证明 (Proof Surrogates) | 暗示有证据但未引述（"显而易见"、"据知情人士透露"） |
| 重复 (Repetition) | 反复重申同一观点使其显得可信 |

### 煽动性修辞 (Demagoguery)

- **他者化 (Otherizing)**：将某群体划分为"非我族类"
- **妖魔化 (Demonizing)**：将某人/群体描绘成邪恶化身
- **煽动仇外 (Fostering Xenophobia)**：利用对陌生人/外国人的恐惧
- **贩卖恐惧/仇恨 (Fear/Hate Mongering)**：激起原始负面情绪

## 分析步骤

1. **扫描心理倾向**：作者是否表现出内群体偏见或确认偏差？文本是否利用读者的恐惧或从众心理？
2. **识别修辞陷阱**：逐句检查，标记替代证明、闪烁其词、贬抑等
3. **剥离修辞**：去除修辞成分（委婉语/粗直语还原为中性词，删去替代证明），看逻辑主干还剩什么
4. **评估意图**：使用这些手段是为了澄清观点，还是在缺乏证据的情况下说服？
