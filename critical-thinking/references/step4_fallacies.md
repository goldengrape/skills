# 第4步：谬误侦测 (Fallacy Detection)

基于《Critical Thinking》第12版 第6、7、8章。

## 第一类：相关性谬误 (Red Herring Fallacies)

前提与结论在该语境下逻辑不相关，引入干扰信息转移注意力。

### 诉诸人身 (Ad Hominem)

通过攻击提出断言的**人**而非断言本身来驳倒观点。

| 变体 | 描述 |
|------|------|
| 人身攻击 | 侮辱对方人格或特征 |
| 处境攻击 | 指出对方因处境而持有观点，因此不可信 |
| 井下投毒 | 在对方发言前就破坏其信誉 |
| 你也一样 (Tu Quoque) | 指责对方自己也做过同样的事 |
| 基因谬误 | 仅因观点来源而否定它 |

### 稻草人 (Straw Man)

歪曲、夸大或错误表述对方立场，使其容易攻击，然后推翻这个被歪曲的立场。

### 虚假两难 (False Dilemma)

极不合理地限制选项，通常只给两个极端选项。

- **完美主义谬误**：不能完美解决就完全拒绝
- **划界谬误**：不能划出精确界限则两者没区别

### 错置举证责任

将证明责任错误推给对方："你不能证明我错，所以我对"（诉诸无知）。

### 乞题 (Begging the Question)

前提预设了结论为真，或前提仅是结论的另一种表述（循环论证）。

### 诉诸情感 (Appeal to Emotion)

用情绪反应代替逻辑论证。类型：愤怒、恐吓、怜悯、嫉妒、阿谀奉承、负疚感。

### 无关结论 (Irrelevant Conclusion)

前提可能支持某种结论，但不支持论证者实际试图证明的那个结论（答非所问）。

## 第二类：归纳谬误 (Induction Fallacies)

前提虽相关，但太弱，无法为结论提供足够的概率支持。

### 概括谬误

- **仓促概括 (Hasty Generalization)**：基于太小样本得出结论
- **以偏概全**：使用特例或异常案例概括整体
- **轶事证据**：用个人故事反驳统计规律

### 意外谬误 (Accident)

将普遍规则强行应用于明显不适用的特定例外。

### 弱类比 (Weak Analogy)

类比事物在关键特征上差异过大。

### 错误的诉诸

- **错误诉诸权威**：引用非本领域专家的话
- **诉诸公众**：大多数人相信的就是真的
- **诉诸惯例**：大家一直这么做所以是对的

### 因果谬误

- **事后归因 (Post Hoc)**：B在A之后发生 → A导致B
- **同行归因 (Cum Hoc)**：A和B相关 → A导致B（忽略共同原因/巧合）

### 滑坡谬误 (Slippery Slope)

无证据地断言某一步必然导致一连串灾难性后果。

### 不可检验的解释

提出的解释在原则上无法被验证或证伪。

## 第三类：形式/语言谬误 (Formal & Language Fallacies)

### 形式谬误

| 谬误 | 无效推理结构 |
|------|-------------|
| 肯定后件 | 如果P则Q；Q；所以P |
| 否定前件 | 如果P则Q；非P；所以非Q |
| 中项不周延 | 三段论中项未涵盖全部类别 |

### 语言谬误

- **模棱两可 (Equivocation)**：同一词在论证不同部分使用不同含义
- **语意双关 (Amphiboly)**：语法结构不清导致歧义

### 整体与部分

- **合成谬误**：对部分为真 → 对整体必为真（"每个球员强 → 球队强"）
- **分解谬误**：对整体为真 → 对部分必为真（"球队强 → 每个球员强"）

### 其他

- **混淆解释与借口**：对行为原因的解释 ≠ 为行为辩护
- **赌徒谬误**：误以为独立事件概率受过去结果影响

## 分析步骤

1. **相关性检查**：前提与结论真的相关吗？是否存在攻击人、歪曲观点、诉诸情感？
2. **强度检查**：样本够大？类比恰当？是否存在因果关系跳跃？
3. **结构与语言检查**：是否存在结构错误？关键术语含义是否变化？
4. **标注**：明确指出谬误名称（中英文对照），引用文中具体语句为证据
